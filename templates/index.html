Ctrl+Coffee - C01-A

What's the
seatuation?

A lightweight scheduling demo that assigns student study slots to library seats and study areas using regret-based greedy allocation.

Algorithm Regret Greedy
Priority Regret × Duration
1. Build requests 2. Rank by priority 3. Check capacity 4. Return schedule
Input style Dynamic forms

Add students, areas, and time slots without rewriting JSON.

Assignment unit Student time slot

Each input slot stays indivisible.

Capacity rule Overlap checked

No area exceeds capacity.
